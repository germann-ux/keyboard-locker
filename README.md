# Keyboard Locker

> Bloquea tu teclado sin complicaciones, desde la bandeja del sistema.

![Windows](https://img.shields.io/badge/Windows-10+-blue?style=flat-square&logo=windows) ![.NET](https://img.shields.io/badge/.NET-10.0-512BD4?style=flat-square&logo=dotnet) ![C#](https://img.shields.io/badge/C%23-14-green?style=flat-square&logo=csharp) ![License](https://img.shields.io/badge/License-MIT-yellow?style=flat-square)

Keyboard Locker es una pequeña aplicación para Windows que permite bloquear y desbloquear el teclado desde la bandeja del sistema. La idea es simple: evitar entradas accidentales del teclado de forma rápida y sin abrir ventanas molestas.

## Características

- **Soporte multiidioma** — Español e Inglés nativos
- **Una sola instancia activa** — Sin conflictos entre procesos
- **Ligera y sin dependencias** — .NET ya viene incluido, nada más qué instalar

## Instalación

Es muy sencillo:

1. Ve a la sección **Releases** y descarga el instalador más reciente
2. Ejecuta `KeyboardLocker-Setup-1.0.0.exe`
3. Sigue los pasos del instalador (es rápido, te lo prometo)

Una vez hecho esto, la aplicación estará lista en tu bandeja del sistema.

---

## Uso

**Desde la bandeja del sistema:**

- El icono aparece junto al reloj (bandeja derecha)
- **Clic derecho** abre el menú de opciones
- **Doble clic** alterna el estado del teclado entre bloqueado y desbloqueado

**Opciones disponibles:**

| Acción | Descripción |
|--------|-------------|
| Bloquear / Desbloquear | Cambia el estado del teclado |
| Cambiar idioma | Alterna entre Español e Inglés |
| Salir | Cierra la aplicación |

## Idiomas

La app está disponible en:

- **Español** — Interfaz completa
- **English** — Full interface

El idioma se puede cambiar desde el menú contextual en cualquier momento, y la selección se guarda automáticamente para la próxima vez.
